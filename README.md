# PathMapper — AI Life Decision Simulator
### USAII Global AI Hackathon 2026 | Challenge Brief 3

## What It Does
PathMapper helps students and early professionals navigate major 
life and career decisions by modeling tradeoffs, surfacing hidden 
assumptions, and showing likely outcomes across 6 dimensions.

## Live Demo
Frontend: https://second-brain-virid-seven.vercel.app

## Features
- Multi-step decision intake form
- AI-powered tradeoff analysis across 6 dimensions
- Radar chart visualization
- Hidden assumption detection
- What-If scenario explorer
- 30/60/90-day action schedule
- Responsible AI safeguards with confidence indicators

## Tech Stack
- Frontend: React + Vite + Tailwind CSS + Recharts (Vercel)
- Backend: Python FastAPI (Render)
- AI: Groq API (llama-3.3-70b-versatile)
